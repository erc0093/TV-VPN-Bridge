# TV VPN Bridge

把 Mac 上已经可用的 VPN / 代理网络，共享给 LG TV 等设备使用。

[English](./README.md)

## 这是什么

TV VPN Bridge 是一个 macOS 工具。

它的作用是：
- 让 Mac 作为网络桥接入口
- 把 Mac 上已经可用的 VPN / 代理网络共享给电视或其他设备
- 帮你减少软路由、刷机、复杂网络配置的折腾成本

它更适合这类用户：
- 已经在 Mac 上用好了 VPN 或代理
- 想在 LG TV 等设备上使用原生流媒体应用
- 不想单独折腾软路由
- 希望用一个桌面工具快速完成共享

## 功能特点

- 共享 Mac 上的 VPN / 代理网络到电视或其他设备
- 支持 TUN 类型的虚拟网络来源
- 一键开始共享 / 停止共享
- 支持激活码
- 内置购买和售后群入口

- 一台运行 macOS 的 Mac
- Mac 上已经可正常使用的 VPN / 代理网络
- 与电视或其他设备匹配的网络连接方式


## 安装说明

首次打开时，macOS 可能会提示安全风险。  
这是因为当前版本可能还没有完成 Apple 签名和公证。

如果遇到无法打开的情况，可以这样处理：

1. 先尝试打开一次应用
2. 打开“系统设置”
3. 进入“隐私与安全性”
4. 找到被拦截的应用并允许打开
5. 再次启动应用

## 使用步骤

1. 打开clash的tun模式
2. 在软件入口网络选择clash启用的虚拟网卡
3. 在软件入口网络选择出口网络（网线或热点）
4. 点一键共享，授权
完成
